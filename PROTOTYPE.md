## 一、页面清单

```

├── 首页/仪表盘（Dashboard）
├── 交易记录页（Transactions）
│   ├── 交易列表
```

## 二、页面原型详细说明

### 2.1 首页/仪表盘（Dashboard）

#### 2.1.1 页面概览

**页面名称：** 首页/仪表盘
**设计目标：** 让用户快速查看本月收支概览、账户余额和近期交易，支持一键快速记账

#### 2.1.2 结构说明（Wireframe Logic）

| 区域 | 元素 | 描述 |
|------|------|------|
| 顶部/导航 | 顶部栏 | 白色背景，高度 56px，左侧显示应用名称，右侧显示设置图标 |
| | 月份切换器 | 居中显示"2026年2月"，左右箭头切换月份 |
| 主体内容 | 收支概览卡片 | 白色卡片，圆角 12px，阴影，显示本月收入/支出/结余 |
| | 账户总资产卡片 | 白色卡片，显示所有账户总资产金额 |
| | 账户快捷入口 | 横向滚动列表，显示各账户名称+余额，点击进入账户详情 |
| | 近期交易列表 | 最近5笔交易，按日期分组，显示图标+分类+金额 |
| 底部/全局 | 快速记账按钮 | 固定底部中央，圆形浮动按钮，直径 56px，"+"图标 |
| | 底部导航栏 | 固定底部，4个Tab：首页、交易、统计、我的 |

#### 2.1.3 交互细节

**状态反馈：**
- 收支概览卡片：支出金额显示红色（#ff4d4f），收入金额显示绿色（#52c41a），结余根据正负显示对应颜色
- 账户快捷入口：点击账户卡片 → 跳转账户详情页，显示该账户近期交易
- 近期交易列表：点击交易项 → 跳转交易详情页；长按 → 弹出操作菜单（编辑/删除）

**跳转逻辑：**
- 点击设置图标 → 设置页
- 点击月份切换器 → 刷新当月数据
- 点击账户快捷入口 → 账户详情页
- 点击近期交易项 → 交易详情页
- 点击快速记账按钮 → 添加交易页（默认支出类型）
- 点击底部导航栏 → 切换对应页面

**空状态：**
- 无交易记录时，近期交易区域显示空状态插画+文案"暂无交易记录，点击下方按钮开始记账"

#### 2.1.4 UI 设计建议

**视觉重点：**
- 收支概览卡片使用渐变背景（支出卡片浅红渐变，收入卡片浅绿渐变）
- 账户快捷入口使用横向滚动，左侧显示"全部账户"入口
- 快速记账按钮使用品牌色（#1890ff），悬浮在底部导航栏上方

**组件推荐：**
- 月份切换器：Ant Design DatePicker.Month
- 收支概览卡片：自定义卡片组件，支持渐变背景
- 账户快捷入口：Ant Design Carousel 或横向滚动列表
- 近期交易列表：Ant Design List，带分组功能
- 快速记账按钮：Ant Design FAB（Floating Action Button）

**尺寸规范：**
- 卡片间距：16px
- 卡片内边距：16px
- 金额字体大小：24px（收支概览）、18px（账户余额）
- 交易项高度：64px

### 2.2 交易记录页（Transactions）

#### 2.2.1 页面概览

**页面名称：** 交易记录页
**设计目标：** 浏览所有交易记录，支持按日期、账户、分类、类型筛选，支持搜索和编辑删除

#### 2.2.2 结构说明（Wireframe Logic）

| 区域 | 元素 | 描述 |
|------|------|------|
| 顶部/导航 | 顶部栏 | 白色背景，高度 56px，左侧返回箭头，居中显示"交易记录" |
| | 筛选栏 | 日期范围选择器、账户下拉选择、分类下拉选择、类型Tab |
| | 搜索框 | 搜索图标+输入框，占位符"搜索备注内容" |
| 主体内容 | 交易列表 | 按日期分组，每组显示日期+当日收支小计 |
| | 交易项 | 左侧图标+分类名称+备注（灰色小字）+右侧金额 |
| 底部/全局 | 加载更多 | 列表底部显示"加载更多"或"已加载全部" |

#### 2.2.3 交互细节

**状态反馈：**
- 交易项：支出金额红色，收入金额绿色，转账/借贷灰色
- 筛选栏：选中状态下拉框高亮显示
- 搜索框：输入时实时过滤列表，无结果时显示"未找到匹配记录"

**跳转逻辑：**
- 点击返回箭头 → 返回首页
- 点击日期范围选择器 → 弹出日期选择器，选择后刷新列表
- 点击账户下拉选择 → 弹出账户选择弹窗
- 点击分类下拉选择 → 弹出分类选择弹窗（一级→二级）
- 点击类型Tab → 切换交易类型（全部/支出/收入/转账/退款/借贷）
- 点击搜索框 → 聚焦输入框，显示虚拟键盘
- 点击交易项 → 交易详情页
- 长按交易项 → 弹出操作菜单（编辑/删除/复制）

**空状态：**
- 无交易记录时，显示空状态插画+文案"暂无交易记录，点击下方按钮开始记账"

#### 2.2.4 UI 设计建议

**视觉重点：**
- 筛选栏使用浅灰背景（#f5f5f5），与白色列表区分
- 日期分组标题使用加粗字体，当日收支小计使用小字显示在右侧
- 交易项使用卡片式布局，点击有按压反馈

**组件推荐：**
- 日期范围选择器：Ant Design DatePicker.RangePicker
- 账户/分类下拉选择：Ant Design Select
- 类型Tab：Ant Design Tabs
- 搜索框：Ant Design Input.Search
- 交易列表：Ant Design List，带分组功能

**尺寸规范：**
- 筛选栏高度：48px
- 搜索框高度：40px
- 交易项高度：64px
- 日期分组标题高度：40px

### 2.3 添加交易页（Add Transaction）

#### 2.3.1 页面概览

**页面名称：** 添加交易页
**设计目标：** 快速录入交易，支持5种交易类型，30秒内完成录入，表单简洁易操作

#### 2.3.2 结构说明（Wireframe Logic）

| 区域 | 元素 | 描述 |
|------|------|------|
| 顶部/导航 | 顶部栏 | 白色背景，高度 56px，左侧取消按钮，居中显示"添加交易"，右侧保存按钮 |
| 主体内容 | 交易类型Tab | 底部固定，5个Tab：支出/收入/转账/退款/借贷 |
| | 金额输入区 | 大号金额显示，数字键盘，支持小数点后两位 |
| | 账户选择 | 下拉选择，显示上次使用账户 |
| | 分类选择 | 点击弹出分类选择弹窗（一级→二级） |
| | 日期时间选择 | 日期选择器+时间选择器，默认当前日期时间 |
| | 备注输入 | 文本输入框，最多100字，显示字数统计 |
| 底部/全局 | 数字键盘 | 自定义数字键盘，0-9、小数点、删除、确认 |

#### 2.3.3 交互细节

**状态反馈：**
- 金额输入区：输入时实时显示，0.00显示为灰色占位符
- 交易类型Tab：选中状态下划线高亮，切换时重置表单
- 账户选择：点击弹出账户选择弹窗，选择后更新显示
- 分类选择：点击弹出分类选择弹窗，先选一级再选二级
- 保存按钮：必填字段未填写时禁用，填写完成后启用

**跳转逻辑：**
- 点击取消按钮 → 返回上一页，不保存
- 点击保存按钮 → 保存交易记录，更新账户余额，返回上一页
- 点击交易类型Tab → 切换交易类型，表单字段动态变化
- 点击账户选择 → 弹出账户选择弹窗
- 点击分类选择 → 弹出分类选择弹窗
- 点击日期时间选择 → 弹出日期时间选择器

**特殊字段逻辑：**
- **转账类型**：显示转出账户+转入账户+金额三个字段
- **退款类型**：显示退款账户+金额+关联原支出（可选）三个字段
- **借出/借入类型**：显示账户+金额+备注三个字段

**表单校验：**
- 金额必填，且大于0
- 账户必填
- 分类必填（转账类型除外）
- 日期时间必填

#### 2.3.4 UI 设计建议

**视觉重点：**
- 金额输入区使用超大字体（48px），居中显示，突出核心操作
- 交易类型Tab使用底部固定，方便切换
- 数字键盘自定义设计，0-9大号按钮，小数点和删除按钮较小

**组件推荐：**
- 交易类型Tab：Ant Design Tabs，底部固定
- 金额输入区：自定义大号数字显示组件
- 账户/分类选择：Ant Design Select
- 日期时间选择器：Ant Design DatePicker + TimePicker
- 备注输入：Ant Design Input.TextArea，限制100字
- 数字键盘：自定义键盘组件

**尺寸规范：**
- 金额输入区高度：120px
- 金额字体大小：48px
- 交易类型Tab高度：48px
- 表单字段高度：56px
- 数字键盘按钮高度：64px

### 2.4 账单导入页（Import Bill）

#### 2.4.1 页面概览

**页面名称：** 账单导入页
**设计目标：** 导入支付宝/微信账单，解析CSV文件，智能匹配分类，预览确认后保存

#### 2.4.2 结构说明（Wireframe Logic）

| 区域 | 元素 | 描述 |
|------|------|------|
| 顶部/导航 | 顶部栏 | 白色背景，高度 56px，左侧返回箭头，居中显示"账单导入" |
| 主体内容 | 平台选择 | 两个卡片：支付宝（蓝色图标）、微信（绿色图标） |
| | 文件选择区 | 虚线边框区域，显示"点击选择文件"或已选文件名 |
| | 解析进度 | 进度条+百分比显示 |
| | 智能匹配结果 | 显示匹配成功/失败数量，失败记录列表 |
| | 预览表格 | 表格展示前10条记录，支持滚动查看 |
| 底部/全局 | 导入按钮 | 固定底部，"确认导入"按钮，全宽 |

#### 2.4.3 交互细节

**状态反馈：**
- 平台选择：点击选中卡片，边框高亮显示
- 文件选择区：拖拽文件到区域或点击选择文件，选择后显示文件名和大小
- 解析进度：显示解析进度条和百分比，解析完成后显示"解析成功"
- 智能匹配结果：匹配成功显示绿色，匹配失败显示红色，点击失败记录可手动选择分类
- 预览表格：表格可横向滚动，支持查看所有字段

**跳转逻辑：**
- 点击返回箭头 → 返回上一页
- 点击平台卡片 → 选中平台，更新文件选择提示
- 点击文件选择区 → 弹出文件选择器，选择CSV文件
- 点击失败记录 → 弹出分类选择弹窗，选择后更新匹配结果
- 点击导入按钮 → 保存所有记录，返回交易记录页

**异常处理：**
- 文件格式错误：显示错误提示"文件格式不支持，请导出正确的CSV格式"
- 文件解析失败：显示具体失败行号，支持跳过错误行继续导入
- 重复记录：检测重复记录（相同日期+金额+账户），弹出提示"检测到N条重复记录，是否覆盖？"

#### 2.4.4 UI 设计建议

**视觉重点：**
- 平台选择卡片使用大图标+文字，点击有按压反馈
- 文件选择区使用虚线边框，拖拽文件时边框高亮
- 智能匹配结果使用进度条+数字展示，失败记录用红色标注
- 预览表格使用紧凑布局，支持横向滚动

**组件推荐：**
- 平台选择卡片：自定义卡片组件
- 文件选择区：Ant Design Upload.Dragger
- 解析进度：Ant Design Progress
- 智能匹配结果：自定义结果展示组件
- 预览表格：Ant Design Table，支持滚动

**尺寸规范：**
- 平台选择卡片高度：120px
- 文件选择区高度：160px
- 进度条高度：8px
- 预览表格行高：40px

#### 2.5.3 交互细节

- 账户总览卡片：总资产金额大号显示，账户数量小号显示

- 点击返回箭头 → 返回上一页
- 点击账户卡片 → 账户详情页
- 长按账户卡片 → 弹出操作菜单（编辑/删除）

- 添加账户按钮使用品牌色（#1890ff），悬浮在右下角

**组件推荐：**
- 账户总览卡片：自定义卡片组件，支持渐变背景
- 账户列表：Ant Design List，卡片式布局
- 添加账户按钮：Ant Design FAB（Floating Action Button）

**尺寸规范：**
- 账户总览卡片高度：120px
- 账户卡片高度：80px
- 添加账户按钮直径：56px

### 2.6 分类管理页（Category Management）

#### 2.6.1 页面概览

**页面名称：** 分类管理页
**设计目标：** 管理收支分类，预设+自定义分类，支持增删改，支持分类排序

#### 2.6.2 结构说明（Wireframe Logic）

| 区域 | 元素 | 描述 |
|------|------|------|
| 顶部/导航 | 顶部栏 | 白色背景，高度 56px，左侧返回箭头，居中显示"分类管理" |
| | 类型Tab | 两个Tab：支出/收入 |
| 主体内容 | 分类列表 | 按一级分类分组，每个一级分类下显示二级分类 |
| | 分类项 | 图标+名称+操作按钮 |
| | 添加分类按钮 | 固定底部右下角，浮动按钮，"+"图标 |
| 底部/全局 | 无 | 无固定底部元素 |

#### 2.6.3 交互细节

**状态反馈：**
- 类型Tab：选中状态下划线高亮，切换时刷新分类列表
- 分类列表：一级分类可展开/折叠，显示/隐藏二级分类

## 四、全局交互规范

### 4.1 CTA（Call to Action）优先级

| CTA名称 | 优先级 | 视觉强调 | 事件 | 目标 | 反馈 |
|---------|--------|---------|------|------|------|
| 快速记账按钮 | 高 | 品牌色圆形浮动按钮 | 点击 | 添加交易页 | 跳转动画 |
| 保存按钮 | 高 | 品牌色主按钮 | 点击 | 保存记录 | 成功提示+返回 |
| 导入按钮 | 高 | 品牌色主按钮 | 点击 | 导入账单 | 进度提示+成功提示 |
| 确认导入按钮 | 高 | 品牌色主按钮 | 点击 | 保存导入记录 | 成功提示+返回 |
| 取消按钮 | 低 | 次按钮 | 点击 | 返回上一页 | 返回动画 |
| 编辑按钮 | 中 | 文字链接 | 点击 | 编辑表单 | 跳转动画 |
| 删除按钮 | 中 | 文字链接（红色） | 点击 | 删除记录 | 二次确认弹窗 |
| 复制按钮 | 中 | 文字链接 | 点击 | 复制记录 | 成功提示 |

### 4.2 表单校验规则

| 字段 | 校验规则 | 错误提示 | 恢复方式 |
|------|---------|---------|---------|
| 金额 | 必填，大于0，最多两位小数 | "请输入有效金额" | 输入有效金额 |
| 账户 | 必填 | "请选择账户" | 选择账户 |
| 分类 | 必填（转账类型除外） | "请选择分类" | 选择分类 |
| 日期时间 | 必填 | "请选择日期时间" | 选择日期时间 |
| 备注 | 最多100字 | "备注最多100字" | 删除多余字符 |

### 4.3 异步操作加载策略

| 操作 | 加载方式 | 说明 |
|------|---------|------|
| 账单解析 | 进度条+百分比 | 显示解析进度，解析完成后显示"解析成功" |
| 数据导入 | 全屏遮罩+加载动画 | 防止重复操作，导入完成后显示成功提示 |
| 统计计算 | 骨架屏 | 统计数据加载时显示骨架屏，加载完成后显示数据 |
| 列表加载 | 底部加载更多 | 滚动到底部时自动加载更多数据 |

### 4.4 极端情况处理

| 场景 | 处理方式 |
|------|---------|
| 超长文本（昵称、备注） | 超过限制长度时截断，显示"..."，点击展开查看完整内容 |
| 空数据 | 显示空状态插画+文案，引导用户进行操作 |
| 网络超时 | 显示错误提示"网络连接超时，请重试"，提供重试按钮 |
| 权限不足 | 显示错误提示"无权限访问"，返回上一页 |
| 余额不足 | 允许余额为负，显示警告提示"账户余额不足" |
| 文件过大 | 显示错误提示"文件过大，请选择小于10MB的文件" |

### 4.5 手势与快捷键

| 手势/快捷键 | 功能 | 适用平台 |
|-------------|------|---------|
